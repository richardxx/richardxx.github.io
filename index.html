I obtained my PhD from HKUST and Bsc from UESTC in computer science. My research interest is using formal approaches to uncover program behaviors for industry-scale software. Specifically, I worked on following topics over past ten years:

  • Pointer analysis theory, algorithm, and application
  • Intermediate program representation
  • Scalable path-sensitive static analysis techniques for bug hunting
  • Methods to collect and use program execution traces for debugging
Now, I'm interested in any automatic tools and methods to improve software development productivity. Our company, Sourcebrella, is believing in automation should take over low creativity programming work.

Our jobs are opening! All talents in program analysis, compilers, reverse engineering, web service, software data mining, and so on are highly welcome. The salary is highly competitive and the working locations can be Hangzhou, Shanghai, and Shenzhen. Write to us if you feel the same passion and want to join our great adventure. We will reply every job inquiry to respect your interest in us.
